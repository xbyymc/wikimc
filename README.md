# 维基MC | WikiMC
维基MC 是面向我的世界玩家的专属生存维基，
收录新手入门、生存进阶、服务器玩法、整合包资源等全套攻略，
免费开放、持续更新、人人可参与完善。

## 📖 网站介绍
- 完整的 MC 生存指南
- 新手入门 → 进阶玩法
- 服务器规则与教程
- 持续更新，全部免费

## 🔗 访问链接
正式站：https://wikimc.cn
预览站：https://xbyymc.github.io/wikimc/

## 📁 内容说明
网站所有内容均存放于 `src/content/docs/` 目录下，支持 Markdown 格式。
你可以提交修改、补充攻略，帮助我们完善维基！

## 🤝 如何参与贡献
- 补充新手攻略、生存技巧
- 纠错现有文档错别字
- 新增服务器教程、玩法介绍
- 提交整合包资源、实用资料

欢迎 Fork 后提交 PR，一起完善维基MC。
