## 📖 项目简介
维基MC 是面向我的世界玩家的专属生存维基，
收录新手入门、生存进阶、服务器玩法、整合包资源等全套攻略，
免费开放、持续更新、人人可参与完善。
网站所有内容均存放于 `src/content/docs/` 目录下，支持 Markdown 格式。
你可以提交修改、补充攻略，帮助我们完善维基！
